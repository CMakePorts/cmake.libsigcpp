Source Packages

The source packages for libsigc++ are available on the Gnome ftp site.

Binary Packages

Whenever possible, you should use the official binary packages approved by the supplier of your operating system, such as your Linux distribution. For instance, Debian Linux and Fedora Linux have official libsigc++ packages.

Requirements

 pkg-config should be used to build software that depends on libsigc++.

libsigc++ is built and tested for a standards-compliant C++ compiler. Luckily, the recent versions of all major C++ compilers are now sufficiently standards-compliant.

libsigc++ is known to build and work with the following compilers:

  • g++ (>= 3.2, including Apple and Cygwin versions)
  • Microsoft Visual C++ .NET 2003
  • Intel C++ compiler
  • SUN Forte C++ CC
  • IBM AIX xlC
  • Tru64 C++
  • IRIX MIPSpro
